The Viorant Hub is a native macOS desktop app, signed and notarized by Apple.

<Note>
  macOS is the supported platform today. Windows and Linux builds are planned.
</Note>

## Download

<Steps>
  <Step title="Get the latest build">
    Download the Hub from [**dl.viorant.ai/mac**](https://dl.viorant.ai/mac). The link
    always resolves to the current release — `Viorant Hub.dmg`.
  </Step>
  <Step title="Open the disk image">
    Double-click the downloaded `.dmg`, then drag **Viorant Hub** into your
    **Applications** folder.
  </Step>
  <Step title="Launch">
    Open Viorant from Applications. Because the build is Apple-notarized, it opens
    without a security warning.
  </Step>
</Steps>

## Staying up to date

The Hub updates itself. When a new version is available it downloads in the
background and prompts you to relaunch — you don't need to revisit the download page.
