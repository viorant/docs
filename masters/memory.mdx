title: "Memory Master"
description: "Persistent memory for agents — coming soon."
---

The **Memory Master** will let you give agents persistent memory — short- and
long-term recall that carries across runs — authored as a first-class asset inside a
[Space](/concepts/spaces), just like prompts, skills, and agents.

<Note>
  Memory is **coming soon**. It appears throughout the Hub as a disabled option today
  and lands in a later release. The other four Masters —
  [Prompt](/masters/prompt), [Model](/masters/model), [Skill](/masters/skill), and
  [Agent](/masters/agent) — are available now.
</Note>
