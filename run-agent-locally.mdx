---
title: "Run an agent locally"
description: "Compose, test, and run an agent on your own machine."
---

Agents run **on your machine** — composed in the Agent Master and executed locally,
with output streamed back live as it runs.

<Steps>
  <Step title="Compose the agent">
    In the **Agent Master**, bind a [Prompt](/masters/prompt), a
    [Model](/masters/model), and any [Skills](/masters/skill) the agent should use.
  </Step>
  <Step title="Give it a task">
    Open the test panel and enter the task or message for the agent.
  </Step>
  <Step title="Run and watch">
    Run the agent. Its steps and output stream back live, including the tools it
    calls along the way.
  </Step>
</Steps>

<Note>
  Everything here runs locally against your own keys. Running an agent **off your
  machine** — deploying it to Viorant Cloud — arrives with the 1.1 release.
</Note>
